# Python Avançado e Machine Learning

Repositório contendo estudos, exercícios e projetos desenvolvidos utilizando Python, Machine Learning, Estatística e Ciência de Dados.

Os notebooks abordam desde regressão linear e métricas de avaliação até sistemas de recomendação, clusterização e detecção de anomalias.

---

# Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- SciPy

---

# Projetos

## Classificação MNIST e Métricas

Arquivo:
- `Classificacao_MNIST_Metricas.ipynb`

Conteúdo:
- Regressão Logística
- Classificação de imagens
- Dataset MNIST
- KFold Cross Validation
- Accuracy Score
- Predição de números pares e ímpares

O projeto realiza classificação utilizando o dataset MNIST, aplicando regressão logística e métricas de avaliação para validar o desempenho do modelo.

---

## Clusterização de Clientes com KMeans

Arquivo:
- `Clusterizacao_clientes_kmeans.ipynb`

Conteúdo:
- KMeans
- Clusterização
- Curva de Elbow
- Segmentação RFM
- Análise de Clientes

Projeto de segmentação de clientes utilizando métricas de Recência, Frequência e Valor Monetário (RFM), aplicando KMeans para identificação de grupos de clientes.

---

## Detector de Anomalias Gaussiano

Arquivo:
- `Detector_anomalias_gaussiano.ipynb`

Conteúdo:
- Detecção de Anomalias
- Distribuição Gaussiana
- Probabilidade
- Métricas de Classificação
- F1-Score
- Confusion Matrix

Implementação de um detector de anomalias baseado em distribuição normal (gaussiana), utilizando treino, validação e teste para avaliação do modelo.

---

## Regressão Multivariada e Pipeline Completo

Arquivo:
- `Regressao_Multivariada_Pipeline_Completo.ipynb`

Conteúdo:
- Regressão Linear
- Pipeline de Pré-processamento

- StandardScaler

Projeto completo de regressão multivariada incluindo limpeza, normalização, encoding e preparação dos dados para treinamento de modelos.

## Regressão, Gradient Descent, Ridge e Lasso
